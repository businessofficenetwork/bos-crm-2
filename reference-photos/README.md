# Reference Photos

Example photos showing what evidence is needed for specific supplement
categories — not tied to any individual claim. Use these to show PMs
what photo their crew needs to take when a supplement request depends
on it (e.g. "get me a shot like this of the valley metal").

Not to be confused with `/test-data`, which holds claim-specific
audit test fixtures (estimates, measurements, photos) for a specific
claim.

## Convention

One folder per category, named for the damage/scope type it documents:

```
reference-photos/
  valley-metal/
    valley-metal-damaged.png
  ridge-cap/
  overhead-door/
```

Add new categories as folders the same way. Keep filenames descriptive
(what it shows), not dated or numbered.
